Consultas Técnicas

Respuestas detalladas sobre la arquitectura y viabilidad de HydroTrace AI.

¿Por qué utilizar el conjunto de datos de 2019 como base?

Utilizamos los registros históricos de 2019 como nuestra base sólida de datos para generar los dashboards interactivos iniciales. Esta referencia nos permite establecer un "consumo normal" antes de las variaciones extremas de la crisis hídrica actual, sirviendo como el fundamento para que nuestros algoritmos identifiquen desviaciones significativas.

¿Qué tan preciso es el sistema contra las extracciones irregulares?

El sistema utiliza modelos como Isolation Forest para identificar discrepancias críticas entre el consumo registrado y la densidad poblacional real. Al localizar estas irregularidades, podemos mitigar tanto las fugas físicas como las extracciones ilegales de manera quirúrgica, transformando el análisis en decisiones rentables.

¿Cómo se integra esta solución con SACMEX?

HydroTrace AI se estructura bajo un modelo SaaS diseñado para optimizar los recursos de organismos como SACMEX. El sistema no reemplaza su infraestructura, sino que la potencia al priorizar inspecciones en zonas de alto riesgo, reduciendo drásticamente el Agua No Contabilizada (ANC) mediante una estrategia preventiva.
